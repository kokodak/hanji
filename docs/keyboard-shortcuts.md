# Keyboard Shortcuts

Shortcuts currently wired up in [`apps/hanji/src/main.rs`](../apps/hanji/src/main.rs).

## Document

| Action | Shortcut |
| --- | --- |
| New document | ⌘ N |
| Open document | ⌘ O |
| Save | ⌘ S |

## Formatting

| Action | Shortcut |
| --- | --- |
| Toggle strong | ⌘ B |
| Toggle italic | ⌘ I |
| Toggle inline code | ⌘ E |
| Insert link | ⌘ K |

## Editing

| Action | Shortcut |
| --- | --- |
| Select all | ⌘ A |
| Copy | ⌘ C |
| Cut | ⌘ X |
| Paste | ⌘ V |
| Undo | ⌘ Z |
| Redo | ⌘ Shift Z |
| Delete previous word | ⌥ Backspace |
| Delete to start of line | ⌘ Backspace |
| Indent list item | Tab |
| Outdent list item | Shift Tab |

## View

| Action | Shortcut |
| --- | --- |
| Toggle file browser | ⌘ Shift E |
